# Cloudini

Write code. Summon a serverless app.

Cloudini is a framework and CLI that generates a complete serverless application
from a structured codebase.

## Status

Early development—not ready for production use.

## Vision

Cloudini aims to:

- Generate a serverless backend from application code
- Provision infrastructure (DB, queue, cache, load balancing) automatically
- Support major cloud providers (AWS, Azure, GCP)
- Integrate with modern frontend frameworks (React, Vue, Angular)

## Planned Features

- CLI for project scaffolding and deployment
- Framework for defining APIs and domain logic
- Terraform-based infrastructure generation
- Auto-scaling by default
- Built-in support for databases and caching (RDS, Redis/Valkey)
